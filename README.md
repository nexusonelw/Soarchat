
# SoarChat

A sophisticated AI conversation platform featuring multi-model collaboration and interactive canvas workflows.

## 🌟 Key Features

- **Canvas Multi-Window**
  - Flexible drag-and-drop window management
  - Tab support for organized conversations
  - Lightweight and fluid interface

- **Multi-AI Integration**
  - Simultaneous conversations with DeepSeek, Gemini, Claude, and ChatGPT
  - Comparative analysis across AI responses
  - AI-to-AI autonomous dialogue capabilities

- **Workflow Orchestration**
  - Import/export workflow configurations
  - Sequential AI window assembly
  - Visual flowchart editor

- **Knowledge Management**
  - Local document embedding
  - Web content capture and storage
  - Personal knowledge graph construction

- **Development Tools**
  - Integrated terminal with Claude Code and Gemini CLI support
  - Custom API client integration
  - Local and remote tool orchestration

## 💡 Advanced Capabilities
